# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be kind. Disagree with ideas, not people.
- Don't harass. Don't dox. Don't be a jerk.
- Maintainers will moderate; instances of abusive behavior will result in temporary or permanent bans.

Report issues to the repo's maintainers via a private GitHub issue or email.
